# Public tool symbols

These local SVGs support the portfolio's public-tool gallery. The inventory combines public tools visible in the supplied workspace screenshot with later user-requested technical additions; brand-specific, environment-specific, and internal launchers were excluded.

Sources:

- Lucidchart, Figma, Jira, Confluence, Slack, Looker, Elastic Stack, Datadog, WordPress, WooCommerce, Shopify, Cursor, AWS, Linear, OpenAI, Claude, Snowflake, GitHub, Postman, BigQuery, TypeScript, Python, and React: Iconify-hosted Simple Icons SVGs.
- Optimizely and Amplitude: Iconify-hosted SVG Logos collection.
- Ontology + graphs: original connected-node SVG drawn for this site; this represents an architecture pattern rather than a branded application.
- Payment gateways: original card-and-routing symbol drawn for this site; it represents an integration capability without claiming a named payment processor.

The files are local so the gallery remains fast and portable. Product names and marks belong to their respective owners. Their presence is a portfolio inventory, not an assertion of certification, partnership, tenure, production scale, or a uniform level of proficiency.
