# Self-hosted portfolio fonts

- `public/assets/fonts/manrope-latin-variable.woff2` — Manrope 400–800 Latin subset, Google Fonts v20.
- `public/assets/fonts/space-grotesk-latin-variable.woff2` — Space Grotesk 500–700 Latin subset, Google Fonts v22.

Both families are distributed under the SIL Open Font License 1.1. The matching
license files are retained in this directory. Font binaries are fetched from the
official endpoints recorded in `scripts/fetch-fonts.ps1`; production pages make
no Google Fonts request.
